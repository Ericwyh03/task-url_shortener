## URL Shortener

A simple URL shortener application built as a technical assessment.

The app allows users to submit a long URL and receive a shortened link, which redirects to the original URL when visited.

## Tech Stack

- Next.js (App Router)
- Supabase (PostgreSQL)
- Vercel (Deployment)

## Features

- Create short URLs

## Getting Started (Local)

1. Clone the repository
2. Install dependencies

```bash
npm install
```

3. Create a .env.local file based on .env.example
4.Run the development server
```bash
npm run dev
```
5. Open http://localhost:3000 in your browser.
